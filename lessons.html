Lessons

Week 1

These are intended to get you started with R and RStudio.

  • Installing R & RStudio download r, rstudio, and other tools for the course
  • The RStudio IDE tips for managing your code, data, and files

Week 2

These exercises will help you understand how to extract pieces of your data in R.

  • Introduction to R Programming an introduction to functions and objects
  • Subsetting R Objects pull out the pieces from any object in r

Week 3

These lessons cover an introduction to ggplot2 for data visualizations, and how to use the R Markdown, a “notebook interface to weave together narrative text and code to produce elegantly formatted output.”

  • Data Visualization with ggplot2 (part 1) an introduction to the grammar of graphics
  • R Markdown in Practice extending r markdown

Week 4

Data are rarely collected in a way that’s fit for analysis or data visualization. In this lesson we introduce dplyr, a package designed to handle “most common data manipulation challenges.”

  • Introduction to Data Manipulation introduction to wrangling data

Week 5

Now that we’ve introduced the grammar of ggplot2, we can build visualizations that display counts and variable distributions.

  • Data Visualization with ggplot2 (part 2) counting things, variable distributions, ridgelines, and violin plots

Week 6

In previous lessons, we used dplyr to manipulate data. In this lesson, we’ll be using the tidyr package to reshape and restructure data. To demonstrate how these packages work together, we’re going to scrape (download) a Wikipedia table of List of largest biotechnology and pharmaceutical companies.

  • Reshaping and Joining Data in R transpose, pivot, split, and combine data
  • Scraping Wikipedia Tables advanced wrangling example

Week 7

This lesson covers intermediate data visualization techniques like plotting trends, adding help-text to your graph, labeling values to stand out, and using reference lines for comparisons.

  • Data Visualization with ggplot2 (part 3) trends, adding text, labeling values, and reference lines

Week 8

tidyr has functions for generate sequences of data (full_seq()), creating an output for each combination of any input (expand_grid()), and completing missing combinations of data (complete()).

  • Advanced Data Transformation and Manipulation dealing with complex data structures

Week 9

This lesson builds on our previous knowledge of ggplot2 and introduces facets (small multiples) with ggplot2::facet_wrap(), ggforce::facet_wrap_paginate(), and geofacet::facet_geo()

  • Data Visualization with ggplot2 (part 4) advanced facetting and small multiples

Week 10

This week we cover our first case study using data from the executed offenders database on the Texas Department of Criminal Justice website.

  • Case Study (Part 1) don’t mess with texas part 1: scrape html tables
  • Case Study (Part 2) don’t mess with texas part 2: iterate and download with purrr
  • Case Study (Part 3) don’t mess with texas part 3: split and wrangle the data
  • Case Study (Part 4) don’t mess with texas part 3: manipulating images with magick

Week 11

Maps are a great way to give context to any data source. In this lesson, we’re going to collect data with the rtweet package and use it to build a map with ggplot2.

  • Intro to building maps in R building maps with ggplot2
