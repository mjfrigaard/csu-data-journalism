Free resources

All of the resources below are available online free of cost. I will do my best to use these as required reading for the course.

General Resources for R and the tidyverse

  • R for Data Science: this is the reference manual for all things R and the tidyverse. Bookmark this and follow the author (Hadley Wickham) on Twitter

  • Hands on Programming with R: an introductory text for non-programmers by Garrett Grolemund–Data Scientist and Master Instructor for RStudio–that walks through multiple examples and some excellent programming principles

  • R for Journalists: A fantastic intro to R course designed for journalists by Andrew Ba Tran. It’s a lot like this course, but with less tidyverse and more base R :)

  • Tidyverse: the official website for the tidyverse and it’s accompanying packages. Features many examples and package vignettes

  • The RViews Community Blog regularly posts new packages, features, and all things R-related.

Data Visualization Resources

  • The ggplot2 book (3rd edition) by Hadley Wickham, Danielle Navarro, and Thomas Lin Pedersen

  • The ggplot2 reference on the tidyverse website

  • Fundamentals of Data Visualization is an excellent reference by Claus Wilke for anything and everything you’d like know about figures and graphs. The text is language-agnostic, but all the graphs and figures were created with ggplot2 and the source code is freely available to use

  • Data Visualization: A practical introduction is an excellent resource for getting started with data visualization and R. I also highly recommend following the author on Twitter and reading his blog

R Markdown Resources

  • This Markdown tutorial by Commonmark is ten-twenty minute tutorial on the markdown syntax

  • R Markdown: The Definitive Guide: A comprehensive but friendly introduction to R Markdown and friends

Resources for Data Manipluation

  • Data Wrangling with R is a great online text for a deeper dive into data wrangling.

  • The tidy data chapter in R for Data Science.

  • RStudio Community posts tagged dplyr

  • RStudio Community posts tagged tidyr

  • Stack Overflow questions tagged with tidyr

  • Stack Overflow questions tagged with dplyr

Advanced R topics

  • Advanced R: a great follow-up text to R for Data Science (by the same author).

  • R Language Definition: a comprehensive user manual.

Advanced Rmarkdown Resources

  • YAML: check out the ymlthis package for tools and documentation for working with YAML

  • The R Markdown Cookbook by Yihui Xie, Christophe Dervieux, Emily Riederer
